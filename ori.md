# 软件需求文档：SFTI科幻人格测试

## 1. 项目背景
SFTI（Sci-Fi Type Indicator）科幻人格测试是一款面向高校科幻社团文化节的小程序，旨在通过趣味性的心理测试，为参与者生成独特的科幻人格身份，并利用AI生成个性化形象卡，提升互动体验和传播性。

## 2. 目标用户
- 校内学生（主要）
- 校外社会人士（次要）

## 3. 产品概述
产品为一款轻量级H5应用（或小程序），用户通过回答4道选择题，从四个维度（技术/人文、自由/秩序、开拓/守护、宏大/微小）确定自己的科幻人格类型，最终生成包含雷达图、人格描述和AI定制形象的结果页，并支持保存形象卡图片。

## 4. 功能需求

### 4.1 测试流程
| 步骤 | 功能描述 |
|------|----------|
| 1. 首页 | 展示产品名称（SFTI科幻人格测试）、科幻风格背景图、一句吸引人的标语（如“解锁你的星际身份”）、一个“开始测试”按钮。 |
| 2. 答题页 | 依次展示4道题目，每题对应一个维度。每道题包含题干和两个选项（代表维度两极）。用户选择后自动进入下一题。支持返回修改。答题完成后进入上传图片页。 |
| 3. 图片上传页 | - 提示用户上传一张个人照片（用于生成专属科幻形象）。<br>- 支持从相册选择或拍照上传。<br>- 提供默认头像选项（用户可跳过上传，使用默认科幻剪影）。<br>- 上传后显示预览图，并有一个“生成我的科幻形象”按钮，点击后调用豆包大模型API生成形象。 |

### 4.2 结果页
| 元素 | 说明 |
|------|------|
| **雷达图** | 四维雷达图，展示用户在技术/人文、自由/秩序、开拓/守护、宏大/微小四个维度上的倾向值（建议用0-100%表示）。雷达图样式需符合科幻风格（如发光线条、网格）。 |
| **人格类型名称** | 根据维度组合自动匹配16种人格之一的名称（如“星河拓荒者”“流浪诗人”）。 |
| **人格描述** | 2-3句话描述该角色的特征、典型场景和价值观。 |
| **生成的形象卡** | 显示由豆包大模型基于用户照片和人格类型生成的科幻风格肖像。 |
| **按钮** | “保存形象卡”按钮：点击后将整个结果区域（包含雷达图、文字、形象）合成为一张图片并保存到用户相册。 |
| **分享按钮** | 可分享结果图片到微信朋友圈或好友。 |

### 4.3 豆包大模型集成
- **调用时机**：用户点击“生成我的科幻形象”后，前端将用户上传的图片（base64或文件）和人格类型代码（如“T-F-E-M”）发送至后端，后端调用豆包API生成形象。
- **API要求**：需与豆包大模型协商接口，输入：用户图片+人格类型提示词（例如“科幻风格肖像，星河拓荒者，技术派，自由意志，开拓者，宏大视野”），输出：生成的图片URL或base64数据。
- **超时与降级**：若API调用失败或超时，使用默认科幻剪影作为备选，并提示用户稍后重试。

## 5. 非功能需求

### 5.1 界面风格
- **科幻主题**：深色背景（深蓝、黑色），辅以霓虹蓝、紫色、青色等发光元素。字体采用无衬线科技感字体（如Orbitron、Rajdhani）。
- **动效**：按钮悬停发光、页面切换的淡入/滑动效果、雷达图绘制动画。
- **响应式**：适配手机屏幕（主要），同时兼顾平板。

### 5.2 性能要求
- 页面加载时间不超过3秒。
- 图片上传和生成应在10秒内完成（考虑API延迟），前端需显示加载中状态。
- 支持并发用户数：预计文化节期间峰值100人同时使用。

### 5.3 兼容性
- 支持主流移动端浏览器（iOS Safari、Android Chrome、微信内置浏览器）。

### 6.2 用户数据存储
- 建议不强制用户注册，结果页生成后即可保存图片，无需后端存储用户数据。如需统计，可记录匿名访问次数和人格类型分布。

## 7. 技术方案建议
- **前端**：Vue.js 或 React 的移动端H5框架，使用Canvas或ECharts绘制雷达图，使用html2canvas实现截图保存。
- **后端**：可选Node.js或Python Flask，用于转发豆包API请求（避免前端暴露密钥），并提供静态数据接口。
- **部署**：使用云服务器或静态托管（如Vercel、Netlify）加轻量级后端。

## 8. 验收标准
- 用户可完整走完答题、上传图片、生成结果流程。
- 结果页正确显示雷达图、人格信息、AI形象。
- 点击保存按钮可成功保存图片到本地。
- 界面在不同手机型号上显示正常，无明显布局错乱。
