## 1. 引言

### 1.1 项目背景
“SFTI科幻人格测试”是一款面向校园科幻社团文化节的小程序，旨在通过趣味性的心理测试，让参与者快速获得一个独特的科幻角色身份，并生成个性化的形象卡用于社交分享。本产品将作为社团摊位的主要互动工具，吸引人流并传播科幻文化。

### 1.2 项目目标
- 提供一套简洁有趣的科幻人格测评，让用户在3分钟内完成测试。
- 基于四个核心维度（技术/人文、自由/秩序、开拓/守护、宏大/微小）生成16种人格类型之一。
- 展示多维度的测评结果，包括雷达图和文字描述。
- 允许用户上传个人照片，利用AI生成专属的科幻风格形象卡。
- 整体界面需呈现强烈的科幻美学，符合科幻社团的定位。

### 1.3 适用范围
本文档适用于开发团队、UI/UX设计师、测试人员，作为后续设计、开发、测试的依据。

## 2. 总体描述

### 2.1 用户角色
| 角色 | 描述 |
|------|------|
| 普通用户 | 校内学生或社会人士，通过社团摊位扫码或直接访问小程序，参与测试并生成结果。 |
| 社团管理员 | 负责维护测试题目、人格类型库，监控小程序运行状态（可选，初期可不实现）。 |

### 2.2 产品范围
本产品为微信小程序形态，核心功能包括：
- 人格测试（答题 + 结果计算）
- 结果展示（雷达图 + 文字描述）
- 形象卡生成（用户上传图片 + AI生成 + 保存）

### 2.3 假设与依赖
- 用户需要授权访问相册/相机以上传图片。
- 需要接入第三方图像生成API（如豆包大模型或其他AI绘画服务），确保生成速度和效果。
- 小程序需部署在稳定服务器上，支持并发访问。

## 4. 非功能需求

### 4.1 性能
- 答题页面切换应无卡顿，响应时间 < 0.5秒。
- 调用AI生成接口的等待时间应在10秒以内（含网络传输和生成），超时需要提示。
- 支持同时在线用户数：预估社团文化节期间峰值并发200人。

### 4.2 安全性
- 用户上传的图片仅用于本次生成，不应持久化存储（如需缓存，需明确告知用户并定期清除）。
- 接口调用需进行身份校验，防止恶意刷接口。

### 4.3 兼容性
- 支持 iOS 和 Android 主流微信版本（微信 7.0 及以上）。
- 适配不同屏幕尺寸（375px ~ 414px 宽度）。

### 4.4 易用性
- 所有操作步骤不超过3次点击。
- 字体大小适中，科幻风格但需保证可读性。

## 5. 界面需求

### 5.1 整体风格
- **主色调**：深空蓝（#0B0F1F）、暗紫色（#1A0F2A）、青色（#00F0FF）作为点缀。
- **字体**：标题使用具有科技感的字体（如“优设标题黑”或“D-DIN”），正文字体使用易于阅读的无衬线体（如PingFang）。
- **元素**：几何线条、发光效果、星空渐变、全息投影质感。

### 5.2 关键页面设计参考
- **首页**：背景为动态星空或静态星云，中间放置科幻风格logo和标题，下方按钮带有扫描线动画。
- **答题页**：题目卡片有半透明毛玻璃效果，选项带有发光边框，选中后变色。
- **结果页**：雷达图采用发光圆环，维度标签带有科幻图标（如扳手、握手、飞船、星星）。
- **形象卡生成页**：加载动画可使用宇宙飞船或星环旋转的SVG动画。

## 6. 数据需求

### 6.1 题目数据表（示例）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | int | 题目编号 |
| question | string | 题干 |
| option_a | string | 选项A内容（对应某个维度极向，如技术） |
| option_b | string | 选项B内容（对应人文） |
| option_c | string | 选项C内容（如自由） |
| option_d | string | 选项D内容（如秩序） |
| dimension_a | string | 选项A对应的维度（如“T”） |
| dimension_b | string | 选项B对应的维度（如“H”） |
| dimension_c | string | 选项C对应的维度（如“F”） |
| dimension_d | string | 选项D对应的维度（如“O”） |

*注：每个题目应同时覆盖两个不同的维度，即选项A和B对应一个维度的两极，选项C和D对应另一个维度的两极。具体需根据题目设计。*

### 6.2 人格类型表
| 字段 | 类型 | 说明 |
|------|------|------|
| code | string | 四字母代码，如“T-F-E-M” |
| name | string | 人格名称（如“未来科学家”） |
| description | string | 详细描述（约100字） |
| short_desc | string | 一句话描述（用于展示） |

### 6.3 结果计算逻辑
- 每个维度独立计分。例如“技术/人文”维度：每选一次技术（T）则技术得分+1，选人文（H）则人文得分+1。最终技术得分占比 = 技术得分 / (技术得分+人文得分) * 100%，若技术得分>人文得分，则结果为技术倾向，反之为人文。若有相等，可预设规则（如默认为技术）。
- 其他三个维度同理。
- 最终根据四个维度的倾向组合，匹配人格类型表中的code（如“T-F-E-M”）。

技术派系列
T-F-E-M 未来科学家
技术 + 自由 + 开拓 + 宏大

实验室的灯光彻夜不灭，你盯着全息屏幕上的数据流，距离突破只剩最后一道计算。没有人给你下达指令，也没有截止日期，但你停不下来——因为你知道，这项发现将把人类文明推向新的纪元。你是那个在未知领域点燃火把的人。

一句话：真理在远处，我走过去。

T-F-E-I 星河拓荒者
技术 + 自由 + 开拓 + 微小

你的飞船不大，设备不算先进，但你熟悉每一个零件的脾气。你专挑那些大舰队看不上眼的边角星系，去采集样本、绘制地图、寻找稀有矿产。赚的不多，但每次发现一颗新的小行星，你都会给它起一个只有自己知道的名字。

一句话：群星那么大，我只要一小块。

T-F-G-M 太空游侠
技术 + 自由 + 守护 + 宏大

你不属于任何势力，但商船遇险时会收到你的信号，殖民地遭袭时你会出现在地平线上。你的飞船改装到了极限，武器比同等尺寸的战舰还多。没人付你薪水，你说，看见求救信号不回应，睡不着觉。

一句话：我是我自己的舰队，我守护我想守护的。

技术 + 自由 + 守护 + 微小

一个人，一座偏远深空监听站，二十年。你每天记录着来自宇宙各处的电磁波，大部分是杂音，偶尔有商船通讯。你在等一个信号——那个据说存在但从未被证实的远古文明的呼唤。没人要求你守在这里，但你怕万一真的来了，没人听见。

一句话：群星沉默，我是那个还在听的人。

T-O-E-M 先遣指挥官
技术 + 秩序 + 开拓 + 宏大

你的舰队永远走在最前面，为新移民开辟安全的航路。每一次跃迁都有详细预案，每一个未知星系都先派无人机侦查，每一场遭遇战都有撤退方案。你的船员说，跟着你，再远的地方都敢去。

一句话：开路的人，不能走错一步。

T-O-E-I 深空测绘员
技术 + 秩序 + 开拓 + 微小

你飞过的星系，都会变成精确的星图。那些危险的小行星带、不稳定的恒星、适合建站的类地行星，都被你一一标注。你的名字永远不会出现在教科书上，但每一艘经过这些星域的飞船，用的都是你绘制的航线。

一句话：我把路标立好，剩下的事交给他们。

T-O-G-M 星球执政官
技术 + 秩序 + 守护 + 宏大

一百二十万居民，七座城市，三套生命维持系统，都在你的管理之下。每天有处理不完的报告、协调不完的部门、应对不完的突发状况。你不认识每一个人，但你知道，他们的安全取决于你做的每一个决定。

一句话：这颗星球的重，我扛着。

T-O-G-I 飞船工程师
技术 + 秩序 + 守护 + 微小

你手上的茧比谁都厚，身上永远有机油的味道。别人关心飞船能飞多快，你关心引擎的第三号管路是不是该换了。你检修过的飞船，从来没在半路出过故障。没人给你颁奖，但船长们都说，这艘船交给你看过，我放心。

一句话：我不开船，我让船能开。

你带着人类的全部记忆——音乐、诗歌、历史、哲学——驶向未知的星系。遇见新文明时，你不急着谈判，先问他们怎么理解爱，用什么方式纪念逝者，有没有一首歌是唱给星星的。你是人类派出去的使者，不是代表，是朋友。

你的博物馆在一艘旧货船上，在各个殖民地之间漂泊。展品是你收集来的：地球时期的照片、某个人手工做的玩具、第一代移民写给故乡的信、快灭绝的植物的标本。你相信，文明不是只有飞船和武器，还有这些被遗忘的小东西。

一句话：我替人类保管那些差点丢掉的东西。

H-O-E-M 文明考古官
人文 + 秩序 + 开拓 + 宏大

你寻找那些已经消失的文明，在废墟里拼凑他们的历史。每一个碎片都要编号、记录、分析，每一处遗址都要绘制三维结构、推测功能、还原生活。有人说，死了的文明有什么好在意的。你说，我们也会死，总得有人记得我们怎么活过。

一句话：给消失的文明，立一块碑。

人文 + 秩序 + 开拓 + 微小

你不关心舰队去了哪里，只关心他们走后留下的土地。盐碱地怎么变良田，污染的水怎么净化，被破坏的生态怎么修复。你有一套完整的方法论，每到一个地方就按步骤推进。几十年后，有人指着那片森林说，这是谁种的？你不回答，只是继续往前走。

一句话：我把荒地变成家，然后交给陌生人。

H-O-G-M 星区代表
人文 + 秩序 + 守护 + 宏大

你是三十二个殖民地共同选出来的人，代表他们在议会发声。每一份议案都要研究，每一个诉求都要回应，每一次投票都要对得起选你的人。你认识每个殖民地的代表，记得他们的名字、他们的困难、他们的希望。权力不大，责任不小。

一句话：他们选了我，我不能让他们失望。

H-O-G-I 殖民地警官

一句话：平安不是没有事，是出了事有人管。

## 8. 风险与应对
| 风险 | 应对措施 |
|------|----------|
| AI生成形象卡速度慢或失败 | 增加备用生成策略，如使用预置模板合成；添加超时重试机制。 |
| 小程序审核被拒 | 确保不收集用户隐私信息，图片仅用于生成且不存储。 |
| 文化节期间流量突增 | 提前进行压力测试，服务器采用弹性扩容。 |
